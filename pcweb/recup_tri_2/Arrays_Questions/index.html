Ordenação de arrays

1. Crie uma função que recebe um array de 2 números e coloque eles em ordem crescente. Não use o método sort.

Elaboração do código em português:

Primeiramente, temos um array contendo números. Logo, torna-se necessário a utilização de uma estrutura de repetição (for) para percorrer todo o array; depois disso, precisa ter um outro array para trocar as posições dos elementos

A ordenação de elementos ocorre da seguinte maneira: verificação se o número anterior é maior do que o número posterior; caso o número anterior for maior, eles trocam de posição e segue sucessivamente.
